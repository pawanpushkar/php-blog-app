# PHP Blog Application

A simple and secure blog application built using **PHP, MySQL, and Bootstrap**.  
This project was developed as part of an internship task to demonstrate backend development, database integration, and basic web security practices.

---

## 🚀 Features

- Display blog posts from MySQL database
- Search functionality for posts
- Pagination for better content navigation
- Responsive user interface using Bootstrap
- Secure database queries using prepared statements

---

## 🔐 Security Enhancements

The following security measures were implemented:

- **Prepared Statements** to prevent SQL Injection
- **Output Escaping (`htmlspecialchars`)** to prevent XSS attacks
- **CSRF Token Protection** for secure form submissions
- **Input Validation** for user input
